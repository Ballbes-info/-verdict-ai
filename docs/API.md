# API Endpoints

## GET /
Главная страница. Возвращает welcome.html.

## POST /
Вход пользователя.
Параметры: email, password

## GET /register
Страница регистрации.

## POST /register
Создание нового пользователя.
Параметры: email, pass

## GET /court
Страница суда (в разработке).

## GET /select_mode
Страница выбора режима.

## GET /about
Страница "О проекте".

## GET /contact
Страница контактов.
