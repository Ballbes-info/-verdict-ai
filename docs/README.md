# BinaryClash

> «Законы придумали люди. Искусственный интеллект доведет их до абсолюта»

Интерактивный AI-симулятор судебных процессов в реальном времени. Виртуальные юристы сходятся в словесных дуэлях, пользователь меняет правила игры на лету.

---

## О проекте

Проект создан после окончания Яндекс Лицея и защиты выпускного проекта ЛиВА (Лицей имени Вадима Аганова). BinaryClash — переход от учебных задач к полноценной игровой ИИ-экосистеме.

В основе платформы — три ИИ-агента:

- Прокурор. Агрессивный, дотошный. Цепляется за детали, давит на защиту.
- Адвокат. Изворотливый, харизматичный. Ищет лазейки, манипулирует фактами.
- Судья. Хладнокровный. Взвешивает аргументы, выносит вердикт.

---

## Ключевые особенности

- Автономия ИИ. Модели ведут спор самостоятельно, опираясь на заложенные психотипы.
- Вмешательства. Игрок может подкупить судью, подбросить улику или заставить сторону сменить стратегию.
- Атмосфера. Посимвольный вывод текста, скрип дверей, удары молотка.
- Экономика жетонов. Активность дает доступ к более продвинутым ИИ-моделям.
- Хаос-режим. Случайные события меняют ход процесса.

---

## Игровые режимы

- Классический суд. Прокурор против Адвоката. Судья выносит решение.
- Человек против ИИ. Игрок выступает адвокатом или прокурором.
- Научная дискуссия. Два ИИ спорят на заданную тему.
- Хаос-суд. Случайные события вмешиваются в процесс.

---

## Технологический стек

- Backend: Python, Flask
- Frontend: HTML5, CSS3, Jinja2, JavaScript
- База данных: SQLite, SQLAlchemy
- ИИ: OpenRouter API, кастомные промпты
- Аутентификация: Flask-Login, Werkzeug
