## Этап 1. MVP для олимпиады (июль–август 2026)

**Цель:** Работающий продукт, который можно показать на НТО и «Больших вызовах».

**Задачи:**
- Страница суда с чатом (игрок пишет → ИИ-прокурор отвечает)
- ИИ-судья, выносящий вердикт
- Лимит 15 сообщений на процесс
- Сохранение истории процессов в базу данных
- Простейший рейтинг (победы/поражения)
- Интеграция с OpenRouter API (бесплатные модели)

**Результат:**
- Ты можешь зарегистрироваться, начать суд, получить вердикт
- Проект работает на бесплатном хостинге
- Готова презентация для олимпиады

**Это соответствует:** ТЗ 26.4 (MVP)

---

## Этап 2. Подготовка к олимпиаде (сентябрь–ноябрь 2026)

**Цель:** Упаковать проект для участия в НТО, «Больших вызовах», перечневых олимпиадах.

**Задачи:**
- Полировка интерфейса (мобильная адаптация)
- Документация: архитектура, API, промпты
- Демо-видео работы проекта (3–5 минут)
- Подготовка презентации (10 слайдов)
- Подача заявок:
  - НТО (профиль «Искусственный интеллект») — регистрация до октября
  - «Большие вызовы» — до 20 февраля
  - Перечневые олимпиады (ИТМО, ВШЭ, МФТИ)

**Результат:**
- Заявки поданы
- Проект готов к защите
- Есть шанс на диплом → льготы при поступлении

**Это соответствует:** ТЗ 26.5 (Closed Alpha) — ограниченное количество тестировщиков

## Этап 3. Расширение функционала (декабрь 2026 – март 2027)

**Цель:** Добавить фичи, которые выделят проект среди конкурентов на олимпиаде.

**Задачи:**
- Механика «Вмешательств» (игрок может подкинуть улику или попросить судью сменить тему)
- Режим «Человек против ИИ» (игрок — адвокат или прокурор)
- Режим «Научная дискуссия» (два ИИ спорят на заданную тему)
- Таблица лидеров с рейтингом ELO
- Экспорт истории суда в текстовый файл

**Результат:**
- Проект стал интереснее для демо
- Больше аргументов для жюри

**Это соответствует:** ТЗ 26.6 (Alpha) — новые игровые режимы

---

## Этап 4. Финальная подготовка к поступлению (апрель–август 2027)

**Цель:** Получить дипломы и предложения о зачислении.

**Задачи:**
- Участие в финалах олимпиад
- Сбор дипломов и грамот
- Подготовка портфолио (GitHub + презентация + видео)
- Если есть диплом → подача документов на БВИ (без ЕГЭ)
- Если диплома нет → поступление через ИД (индивидуальные достижения) + ЕГЭ

**Результат:**
- Поступление в топовый вуз (ИТМО, ВШЭ, МФТИ, Бауманка)

**Это соответствует:** ТЗ 26.7–26.8 (Closed/Open Beta)

## Что важно понимать

Этот план — сокращенная версия пункта 26 из ТЗ (26.1–26.14). Полная версия описывает проект уровня стартапа с командой 10+ человек и бюджетом 10+ млн рублей.

Мы делаем упрощенную версию, но строго по той же логике: от концепции к MVP, от MVP к тестированию, от тестирования к релизу.

**Разница только в масштабе.**
